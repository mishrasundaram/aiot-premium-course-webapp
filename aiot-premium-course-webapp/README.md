# AIoT Premium Course Web App

Premium Next.js landing page for the course: **AIoT: Artificial Intelligence of Things — Edge AI, TinyML & Smart Systems**.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Deploy

Push the project to GitHub, then import the repository in Vercel. Vercel auto-detects Next.js projects.
